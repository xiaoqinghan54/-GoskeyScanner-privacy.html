Политика конфиденциальности

Последнее обновление: Март 2026 г.

Настоящая политика описывает, как приложение «Gosuslugi Helper» обрабатывает данные пользователей. Мы уделяем первостепенное внимание вашей безопасности и конфиденциальности.

1. Сбор и хранение данных

Приложение не собирает, не передает и не продает третьим лицам никакие персональные данные. Вся информация, вводимая пользователем (номера документов, имена, даты), хранится исключительно на вашем устройстве.

Для защиты данных используется алгоритм шифрования AES-256. Мы не имеем доступа к вашей информации.

2. Использование модуля NFC

NFC используется только для чтения чипа электронного заграничного паспорта. Считанные данные отображаются только на экране устройства. Они никогда не сохраняются в постоянную память и не передаются по сети.

3. Доступ к камере

Камера используется исключительно для сканирования документов с целью их последующего сохранения в формате PDF. Все созданные файлы хранятся локально в защищенной песочнице приложения.

4. Сторонние сервисы

Приложение может открывать сторонние приложения (например, «Госуслуги») для подписания документов. Передача данных не происходит автоматически — пользователь сам инициирует передачу файла через стандартное системное меню «Поделиться» (Share Sheet).

5. Безопасность и ключи шифрования

Ключи шифрования хранятся в защищенном хранилище iOS Keychain с параметром защиты kSecAttrAccessibleWhenUnlockedThisDeviceOnly. Это гарантирует, что данные доступны только тогда, когда устройство разблокировано, и они не копируются в облачные резервные копии без вашего ведома.

6. Конфиденциальность детей

Приложение не предназначено для детей младше 13 лет. Мы намеренно не собираем информацию о несовершеннолетних пользователях.
